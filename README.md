# Introduction

## ExDirectUI

ExDirectUI（Exdui）引擎是一款采用DirectUI概念，基于GDI/Direct2D绘图引擎的轻量级用户界面引擎；它相对其他轻量级UI引擎具有高效、安全、绚丽、扩展性强、可跨语言等特点。

[开始阅读](https://docs.exdui.org)
