## 声明

|动态库命令| 返回值类型|库文件名|参数量| 备注|
|:--:|:--:|:--:|:--:|:--:|
| _img_copyrect |  整数型 |  libexdui.dll | 6 | 复制指定区域图像，成功返回0 |

## 参数列表

| 参数名 |  类型  | 传址 | 数组 | 可空(NULL) |      备注      |
| :----: | :----: | :--: | :--: | :--------: | :------------: |
|  hImg  | 整数型 |  -   |  -   |     -      |    图像句柄    |
|   x    | 整数型 |  -   |  -   |     -      |    横向坐标    |
|   y    | 整数型 |  -   |  -   |     -      |    纵向坐标    |
| width  | 整数型 |  -   |  -   |     -      |      宽度      |
| height | 整数型 |  -   |  -   |     -      |      高度      |
| dstImg | 整数型 |  √   |  -   |     -      | 返回新图像句柄 |


# 示例

## 易语言

```basic
如果 (_img_copyrect(hImg,x,y,width,height,dstImg) ＝ 0)
	输出调试文本(dstImg)
否则
	输出调试文本("复制失败")
```
