##### ExDirectUI Menu Flags

---

### EMNF

| Name               | Value       | Meaning  |
| --                 | --          | --       |
| EMNF_NOSHADOW      | 0x80000000  | 不显示菜单阴影  |
| EMBF_WINDOWICON    | 0x80000000  | 显示窗口图标  |
| EMBF_CENTEWINDOW   | 0x40000000  | 消息框居父窗口中间  |
| EMBF_SHOWTIMEOUT   | 0x20000000  | 显示倒计时  |
